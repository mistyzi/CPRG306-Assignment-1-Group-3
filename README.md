This project is a responsive unit‑conversion website built using JavaScript, Tailwind CSS, and a higher‑order function that performs metric ↔ imperial conversions.

The website supports:
- Weight conversions (pounds ↔ kilograms)
- Distance conversions (miles ↔ kilometres)
- Temperature conversions (Celsius ↔ Fahrenheit)
- Single values and arrays of values
- Responsive Tailwind UI with three tabs
